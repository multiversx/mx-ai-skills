# MultiversX SDKs

## Core
- **mx-sdk-rs**: Rust SDK (Off-chain tools, interacting with chain).
- **mx-sdk-js**: JavaScript/TypeScript SDK (Core logic).
- **mx-sdk-go**: Golang SDK.
- **mx-sdk-py**: Python SDK.

## Frameworks
- **sdk-dapp**: React library for building DApps (wraps `sdk-js` + UI components).
- **sdk-nestjs**: NestJS utilities for microservices (API Network Provider, Caching).
- **mxpy**: CLI tool (Python) for contract interactions, deploying, etc.

## Tools
- **xSuite**: Alternative toolkit (JS/TS) for building/testing contracts.
